# 🎵🧱 BlockVerse

A colorful, musical Minecraft/Roblox-style browser game built with React + Vite.

## Features (Step 1 - Lobby)
- 🏠 Animated lobby with floating music note particles
- 🎨 Full avatar customizer (skin, shirt, pants, hats, accessories)
- 🪙 Coin system — spend coins to unlock premium cosmetics
- 🎮 5 game portals: Obby Rush, Beat Sandbox, Crystal Mine, Beat Battle, Style Stage
- 🎵 Musical DNA throughout — animated equalizer bars, shimmer effects, pulsing coins

## Coming Next
- Obby Rush 🏃 — obstacle course with rhythm checkpoints
- Crystal Mine ⛏️ — dig ores to the beat
- Beat Sandbox 🧱 — build worlds with musical blocks
- Beat Battle ⚔️ — fight enemies in sync
- Style Stage 👗 — outfit contest with community ranking

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
